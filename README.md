**<u>Project Overview</u>**

This project demonstrates an end-to-end data analytics workflow using **MySQL**, **Excel**, and **Power BI**.  
The objective is to analyze the Global Superstore dataset and uncover insights on:

- Sales performance  
- Profitability trends  
- Customer purchasing behavior  
- Shipping efficiency  
- Regional performance  

The final deliverables include SQL scripts, cleaned datasets, and an interactive Power BI dashboard.

An interactive Power BI dashboard can be found [here](https://www.dropbox.com/scl/fi/jahcylug146dr6vilfilw/superstore.pbix?rlkey=3av2lpheo5bxq0dwyf5f9i2fd&st=npavgc7e&dl=0)

The SQL queries can be found [here](https://www.dropbox.com/scl/fi/rk43d4w9s7sd0d7syljnx/superstore.sql?rlkey=x5vv7kwcnhmhlhouusyduxxt6&st=b4wqkee9&dl=0)

**Key Skills Demonstrated**

- **Excel Exploratory Analysis**
  - Pivot tables for initial exploration  
  - Trend and summary charts  
  - Quick data validation and data profiling  

- **SQL Data Cleaning & Transformation**
  - Importing raw CSV data into MySQL  
  - Handling data type conversion (dates, numeric fields, etc.)  
  - Joins, aggregations, window functions  
  - Profitability and regional performance queries  

- **Power BI Data Modeling & Visualization**
  - KPI cards (Sales, Profit, Profit Margin)  
  - Sales & profit trend analysis  
  - Regional and category performance  
  - Shipping performance analysis  
  - DAX measures & calculated columns  
  - Slicers, drill-throughs, and interactive filters
